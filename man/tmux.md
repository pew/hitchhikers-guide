# tmux

tmux!

## scroll in tmux

1. `ctrl + b` + `[`
2. arrow keys or `shift` + `page up/down`
3. `control` + `c` (or `q`) when done

## panes

open another pane:

```
ctrl b + "  split vertically
crtl b - %  split horizontally
```

close pane:

```
crtl b + x
```

switch between panes:

```
crtl b + left
crtl b + right
crtl b + up
crtl b + down
crtl b + o  go to next pane
crtl b + ;  go to the last previously used pane
```
